# Termux SMS

Send text message in termux.

## Installation and Setup

> pkg install git -y

> git clone (copy and past your git link)

> cd Termux-SMS

> chmod +x setup.sh

`It is recommended to run setup.sh first`

__Two Ways To Run Setup.sh__

> bash setup.sh

Or

> ./setup.sh


---

__Two Ways Run the Script:__

> bash TermuxSMS.sh

Or

> ./TermuxSMS.sh
